Setting

添加主题

一般使用Solarized

  1. Downloads theme
  2. 文件->偏好设置->打开主题文件夹
  3. 解压缩，放进去，注意文件夹和一个css文件

修改字体

在主题文件夹中，打开相应主题的css文件，将所有font-family属性原内容前加内容方正行黑简体,

Install

快捷键

无序列表：- + space
有序列表：num + . + space
任务列表：-space[space]space
标题：ctrl+数字
表格：ctrl+t
生成目录：[TOC]按回车
选中一整行：ctrl+l
选中单词：ctrl+d
选中相同格式的文字：ctrl+e
跳转到文章开头：ctrl+home
跳转到文章结尾：ctrl+end
搜索：ctrl+f
替换：ctrl+h
引用：> + space or ctrl + shift + q
著重关键字：ctrl + shift + 反引号键（tab 键上面那个键）
代码块：ctrl+alt+f
加粗：ctrl+b
倾斜：ctrl+i
下划线：ctrl+u
删除线：alt+shift+5
插入图片：直接拖动到指定位置即可 or ctrl+shift+i
插入链接：ctrl+k
隐藏or显示侧边栏：ctrl+shift+l
分割线：--- + enter
定义脚注：文字[^脚注]
高亮显示：==文字==
单行换行：shift+enter

双行换行：enter

安装 go 语言开发环境 (Ubuntu)
